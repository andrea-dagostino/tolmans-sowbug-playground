?
Satisfaction Level
Plots a rolling average of combined satiety across hunger, thirst, and temperature. It is a compact proxy for recent drive fulfillment (higher means the agent has been satisfying needs more consistently).
Resource Gathering
Tracks how efficiently the agent gathers resources over time. The solid line shows cumulative resource consumptions (drive-reducing visits). The faded line shows a rolling rate (consumptions per 20 ticks). A steeper cumulative slope indicates faster resource gathering.
Memory Decay
Visualizes the Ebbinghaus forgetting curve per stimulus type. Each solid line shows mean memory strength for that stimulus (Food, Water, Light, Heat). Count (orange, dashed) shows total entries normalized to peak — drops indicate eviction events. Visited (gray, dotted) shows mean spatial familiarity across visited cells.
Cumulative Reward (DQN)
Shows the running sum of exact per-tick rewards emitted by the DQN agent. A rising curve means the policy is improving expected outcomes. Only visible when using the DQN agent.
Reward Components (DQN)
Decomposes each tick reward into drive reduction, shaping, urgency penalty, off-target penalty, and urgent-search terms (bonus for novelty plus penalties for staying/looping when an urgent target is still unknown).
Cognitive Map Replay
Shows which stimuli the agent remembers at each grid position. Cell color indicates stimulus type; opacity indicates memory strength. Hover over the Satisfaction chart to scrub through memory history. The orange dot marks the agent's position at that tick.
